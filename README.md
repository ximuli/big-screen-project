# 适配大屏

保持页面容器元素的宽高比和设计图的宽高比一致，且始终保持页面容器元素水平、垂直居中。

# 布局

使用 Flex 和 Grid 布局。

* 理由1：大屏一般自主可控，可以使用 Chrome 浏览器
* 理由2: Grid 布局方便快捷，行、列、间隙均可自由控制
